Github Link


In this project we followed the agile methodolgy which consisted of three sprints in the interval of three weeks each which have been explained as follows:

Sprint I
Developed the user signup and login modules along with the user authentication module started implementing the database module. Also in this sprint we designed the view in which which the user can look at the the different location where the real estate agent has different properties situated.

Sprint II
In this sprint we made a few UI changes along with incorportation or Google Maps for locating the property. We designed the view in which the user can look at the listed apartments at the particular location.

Sprint III
In this sprint we implemented the designed Location view and Property View along with the designing and implementation of the owner view.
